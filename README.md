# Esme Nicholson Portfolio

A personal portfolio website showcasing the work and projects of Esme Nicholson.

## About

This repository contains the source code for Esme Nicholson's portfolio website.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/johnerikjordan/esme-nicholson-portfolio.git

# Navigate to the project directory
cd esme-nicholson-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
